# curricuForge-Gen-AI-Powered-Curriculam-Design-System

CurricuForge is an intelligent curriculum generation platform that automatically creates structured, industry-aligned learning plans for a wide range of technical courses.

It allows users to generate week-wise or day-wise curricula with clearly defined topics, subtopics, and reference links, and download the complete curriculum as a professionally formatted PDF.

This project is designed to help students, educators, trainers, and institutions quickly create high-quality course structures without manual planning.

Features
• Supports 50+ technical courses
• Week-wise and day-wise curriculum generation
• Well-structured topics and subtopics
• Industry-relevant reference links
• Download curriculum as PDF
• Simple and clean Streamlit interface

Tech Stack
Python
Streamlit
ReportLab
GitHub

How to Run Locally
pip install -r requirements.txt
streamlit run curriculum.py
